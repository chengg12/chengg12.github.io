javascript console 函数使用

发表于 2018-07-19 | 分类于 前端
字数统计: 531 | 阅读时长 ≈ 3

在初学javascript的时候调试一直用的alert，但是随着学习的深入alert已经越来越不够用了。

按下F12就能看到Console的窗口，用来显示网页加载过程中产生的各类信息。

五种常见方法

最简单的就是console.log(),直接显示内容。

一般信息 console.info()、除错信息console.debug()、警告提示console.warn()、错误提示console.error()

console的占位符

字符(%s),整数(%d或%i),浮点数(%f),对象(%o),样式css

分组显示

console.group() 和 console.groupEnd()

显示一个对象所有的属性和方法

console.dir()

显示网页某节点包含的Html/xml代码

console.dirxml()

追踪函数的调用轨迹

console.trace()

计时

console.time() 和 console.timeEnd()

性能分析

console.profile()

更骚的console

现在console控制台的输出是支持css和图片的
