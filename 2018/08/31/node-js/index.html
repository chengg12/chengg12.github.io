node.js

发表于 2018-08-31 | 分类于 服务器，后端
字数统计: 353 | 阅读时长 ≈ 1

在了解到nide.js之前我是先知道了Hexo,在搭建自己的博客的时候对node.js产生了兴趣，当时正在学习php。

学习php的时候我用的是phpstudy,服务搭起来很简单。但是之后在linux系统上搭建服务时才发现原来还有那么多需要配置的东西。Apche或者Nginx开启Http服务器，mod_php5和php-cgi处理客户端的请求响应。

但是使用node.js直接就可以实现整个Http服务器。

开始

作为一个菜鸟，任何事物对我来说从零开始自学都是十分困难的，第一次配置php环境的时候用的phpstudy，这对我来说已经是比在windows server上配置站点文件简单多了。但是node.js直接再server.js文件里配置监听的端口，以及命令行，对我来说都很有趣。

只要下载安装node.js,然后试试在cmd或者终端中node -v //查看node版本

一切没错，新建一个server.js

运行 node server.js 然后浏览器访问 localhost:8001 就可以看到 Hello world!了
